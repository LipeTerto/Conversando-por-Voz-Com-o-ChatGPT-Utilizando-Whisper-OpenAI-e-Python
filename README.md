# Conversando por Voz Com o ChatGPT Utilizando Whisper (OpenAI) e Python

- Este projeto é um desafio proporcioando pelo Bootcamp da DIO - Bradesco - GenAI & Dados

Neste projeto dominamos o desenvolvimento de sistemas que unem as tecnologias de Speech-to-Text e Text-to-Speech, proporcionando conversas multi-idiomas por voz. Em sinergia com o ChatGPT, o Whisper viabiliza transcrições e traduções em diversas línguas, pavimentando o caminho para soluções de comunicação ágeis e adaptativas. Com a combinação do modelo Whisper e da API do ChatGPT, ambos da OpenAI, nossa solução consegue compreender perguntas feitas por voz e responder de forma precisa. E como diferencial, integramos o Google Text-To-Speech (gTTS) para trazer a resposta do ChatGPT à vida por meio da voz.
